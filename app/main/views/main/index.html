Opal-Phaser:

Make browser games in Ruby

Opal-Phaser is an Opal wrapper for the Phaser framework, to write Canvas or WebGL games in Ruby.

Follow Opal-Phaser development and news on GitHub or Twitter, and feel free to chat with us in our Gitter room.

Getting Started

RubyGem Installation

To get Opal-Phaser as a gem, run this in your project directory:

$ gem install opal-phaser

Or, add this line to your project's Gemfile:

gem 'opal-phaser'

You can also specify a Git(Hub) repository, since Opal-Phaser is under development and the version on RubyGems is not always up to date.

gem 'opal-phaser', github: 'orbitalimpact/opal-phaser'
