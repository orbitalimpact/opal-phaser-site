Opal-Phaser:

Make browser games in Ruby

Opal-Phaser is an Opal wrapper for the Phaser framework, to write Canvas or WebGL games in Ruby.

Follow Opal-Phaser development and news on Twitter or GitHub.

Getting Started

To get Opal-Phaser, run this in your project directory:

$ gem install opal-phaser

Or, add this line to your Gemfile:

gem 'opal-phaser'

You can also specify a Git(Hub) repository, since Opal-Phaser is under development and the RubyGems library is not always up to date.

Use either of these lines to do so:

gem 'opal-phaser', git: 'https://github.com/orbitalimpact/opal-phaser.git'

gem 'opal-phaser', github: 'orbitalimpact/opal-phaser'
