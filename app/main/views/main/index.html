Opal-Phaser:

Make browser games in Ruby

Opal-Phaser is an Opal wrapper for the Phaser framework, to write Canvas or WebGL games in Ruby.

Follow Opal-Phaser development and news on Twitter or GitHub.

Getting Started

To get opal-phaser, run this in your project directory:

$ gem install opal-phaser

Or, add this line to your Gemfile:

gem 'opal-phaser'

You might also want to specify a git repository, since Opal-Phaser is under development and the RubyGems library is not always up to date.

You can use either of these lines to do so:

gem 'opal-phaser', git: 'https://github.com/orbitalimpact/opal-phaser.git' gem 'opal-phaser', github: 'orbitalimpact/opal-phaser'

Examples

There is an in-progress collection of examples for Opal-Phaser, which are Ruby versions of Phaser's examples.

You can check out Opal-Phaser examples under the tab above or in their GitHub repository.
